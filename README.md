# Project Description: Build a two-player Tic-Tac-Toe game that runs in the terminal. --
#  -- Two players take turns entering their moves until one of them wins or the game ends in a draw.
# Instructions to run the program: The program is extremely simple. 
# 1. Run the code 
# 2. Player X will go first, the user "X" must input the row number they would like to use first using numbers (Integers only) 0 through 2 
# ---> The board will have been displayed at this point with numbers indicating row and column spots, so chosing row and column numbers is easy 
# 3. Player "X" will then have to repeat step 2, this time for the column choice instead of the row choice.
# ---> The board will now display with Player "X"'s marker and move
# 4. Player "O" will then have to go and repeat the same steps 2 and 3 to place their marker.
# ---> Upon a win, a dialogue will appear indicating the player that won.
# ---> If there happens to be a draw, a different dialogue will display indicating upon this condition
# ---> After one of these two dialogues is displayed, A tally of the total and updated score is displayed
# 5. The program will prompt you with a question of whether or not you want to play again. You must input "yes" or "no"
#  --> If you chose to play again, the steps 1-5 must be repeated.
# ---> If you chose to NOT play again, the code finishes running and you must run the code again in order to play again. This will restart the running score of any previous games.
